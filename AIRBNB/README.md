# 🏠 Airbnb Data Analysis Dashboard | Tableau

## 📌 Overview

This project analyzes Airbnb listing data using an interactive Tableau dashboard. It provides insights into pricing trends, property types, room types, and regional performance to better understand the Airbnb market.

## 🚀 Key Features

* 🏘️ Property type and room type analysis
* 📊 Region-wise availability and ratings
* 💰 Price analysis based on location (zipcode)
* 📈 Year-wise price trends
* 🎛️ Interactive filters (property type, room type, year)

---

## 💡 Key Insights

* Manhattan and Brooklyn dominate listings
* Entire homes/apartments are most preferred
* Pricing varies significantly by location
* Ratings are consistently high across regions
